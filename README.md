🛒 E-Commerce - Web Application

ระบบร้านค้าออนไลน์ (E-Commerce) ที่พัฒนาด้วย Django Framework (Python) รองรับการทำงานแบบครบวงจร ตั้งแต่การสมัครสมาชิก การเลือกซื้อสินค้า การจัดการตะกร้า (Cart) ไปจนถึงการชำระเงินออนไลน์ผ่าน Stripe Payment

👤 User Roles

ระบบแบ่งการใช้งานออกเป็น 2 ส่วนหลัก ได้แก่

👤 ฝั่งผู้ใช้งาน (Customer)
สมัครสมาชิก และเข้าสู่ระบบ
เลือกดูสินค้าและรายละเอียดสินค้า
เพิ่มสินค้าเข้าสู่ตะกร้า (Cart)
ชำระเงินออนไลน์ผ่าน Stripe
ตรวจสอบประวัติการสั่งซื้อ (Order History)

🛠️ ฝั่งผู้ดูแลระบบ (Admin)

ผู้ดูแลระบบสามารถจัดการข้อมูลผ่าน Django Admin Panel ได้ ดังนี้:

เพิ่มสินค้า (Create Product)
แก้ไขข้อมูลสินค้า (Update Product)
ลบสินค้า (Delete Product)
อัปโหลดและจัดการรูปภาพสินค้า
กำหนดราคา และจำนวนสินค้า (Stock)
เปิด / ปิดสถานะการจำหน่ายสินค้า (Available)
